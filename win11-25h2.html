🧹 Windows 11 25H2: Menos Bloatware, Más Control

Microsoft lo intenta de nuevo. Con la actualización de Windows 11 25H2 (esperada para finales de 2025), la compañía planea abordar una de las quejas más comunes de los usuarios: demasiadas aplicaciones preinstaladas que no se pueden eliminar fácilmente. Esta vez, habrá una forma oficial de decir adiós a aplicaciones como Media Player, Paint, Cortana o el Bloc de Notas, sin usar PowerShell ni herramientas de terceros.

Paso 1: Abre el Editor de Políticas de Grupo

La principal novedad llega a través del Editor de Políticas de Grupo. Si usas Windows 11 Pro, simplemente presiona Win + R, escribe gpedit.msc y presiona Enter.

En el Editor de Políticas de Grupo:

  • Ve a Configuración del equipo → Plantillas administrativas → Componentes de Windows.
  • Busca la nueva opción llamada Quitar aplicaciones preinstaladas.
  • Haz doble clic en ella.
  • Selecciona Habilitada.
  • Guarda los cambios.
  • Reinicia tu equipo. ¡Listo! Ahora puedes desinstalar aplicaciones como Paint, Media Player, Cortana, Sticky Notes, etc., directamente desde Configuración.

Si usas Windows 11 Home, desafortunadamente, el Editor de Políticas de Grupo no está disponible oficialmente. Existen métodos no compatibles para habilitarlo o editar el registro, pero no se recomiendan para principiantes.

⚙️ ¿Cómo funciona?

El proceso es simple para los usuarios de Windows 11 Pro. Simplemente usa el Editor de Políticas de Grupo y habilita la opción para eliminar aplicaciones preinstaladas.

Configuración del equipo > Componentes de Windows > Implementación de paquetes de aplicaciones

Para los usuarios de Windows 11 Home, todavía no hay una solución oficial. Se recomienda esperar futuras actualizaciones o usar métodos alternativos con precaución.

Nota: La disponibilidad de esta opción puede variar según tu compilación de Windows 11. Asegúrate de estar actualizado.

Después de habilitar la política, ve a Configuración → Aplicaciones → Aplicaciones instaladas y elimina las aplicaciones que desees.

Recuerda: no todas las aplicaciones se pueden eliminar, pero la lista será más larga que nunca.

🔔 Notas importantes

  • No todas las aplicaciones se pueden desinstalar. Algunos componentes esenciales del sistema permanecerán protegidos.
  • Aplicaciones como Paint, Media Player, Cortana, Sticky Notes y algunas otras se pueden eliminar. Las aplicaciones del sistema como el Explorador de archivos, Microsoft Store o Configuración permanecerán protegidas.
  • La disponibilidad puede variar según la compilación y la región.
  • Siempre haz una copia de seguridad antes de eliminar aplicaciones importantes.

🗓️ ¿Cuándo estará disponible?

La nueva función se espera en la actualización de Windows 11 25H2, programada para finales de 2025. Los usuarios del programa Insider pueden obtenerla antes.

❓ ¿Por qué es importante?

  • Menos bloatware significa un sistema más limpio, rápido y con más espacio libre.
  • Ofrece a los usuarios más control sobre lo que permanece instalado en su PC.

🔎 Más detalles

  • Además de las aplicaciones mencionadas, otras pueden ser eliminables en el futuro a medida que Windows evolucione.
  • La lista de aplicaciones eliminables puede crecer en futuras actualizaciones.
  • Las herramientas de terceros aún pueden ser útiles para una eliminación avanzada.
  • Microsoft puede revertir o cambiar esta política en futuras compilaciones.

Mantente atento a las noticias y siempre lee las notas de la versión antes de actualizar.

✅ Conclusión

La actualización de Windows 11 25H2 marca un paso importante en el control del usuario sobre el sistema.

Eliminar el bloatware nunca ha sido tan fácil, ¡y oficial! Disfruta haciendo Windows tuyo.
