BCDeX: Cómo Corregir el Error "<ProductKey>" en WinPE

Al utilizar BCDeX para instalar Windows, es posible que te encuentres con un mensaje de error que indica "<ProductKey>" al intentar iniciar el proceso de instalación desde el entorno de recuperación de Windows (WinPE). Este problema, aunque puede parecer intimidante, tiene una solución sencilla. A continuación, te explicamos por qué ocurre y cómo puedes corregirlo paso a paso.

¿Por qué aparece el error "<ProductKey>"?

Este error suele aparecer cuando el sistema no puede encontrar o acceder correctamente a la clave de producto de Windows durante la fase inicial de instalación en WinPE. Aunque BCDeX prepara el entorno de instalación de forma automática, en ocasiones, la ruta o el acceso a los archivos de configuración pueden verse afectados, impidiendo que el instalador de Windows detecte la clave de producto necesaria para continuar.

Solución Paso a Paso

Para corregir este problema, simplemente necesitamos indicarle al instalador de Windows dónde encontrar el archivo de configuración que contiene la clave de producto. Sigue estos pasos:

  1. Cuando aparezca el error <ProductKey>, presiona las teclas `Shift + F10`.
  2. En la ventana de la consola de comandos (CMD), escribe `notepad` y presiona Enter.
  3. Esto abrirá el Bloc de Notas.
  4. En el Bloc de Notas, ve a Archivo > Abrir.
  5. En la ventana que se abre, dirigete hasta la partición que creó la BCDeX, la que tiene la etiqueta o nombre BCDeX Vol.
  6. Teniendo seleccionado en "Tipo" Todos los archivos, selecciona el archivo autounattend.xml y presiona "Abrir".
  7. Con el archivo abierto, busca el bloque que dice: "<UserData>".
  8. Dentro de ese bloque, localiza la sección "<Key></Key>" y escribe una clave de producto genérica según tu edición de Windows (Pro, Home, Enterprise, etc.). En este ejemplo se usa una clave para Windows Pro.

  9. Después de introducir la clave de producto en la sección "<Key></Key>", guarda los cambios del archivo.
  10. Cierra el Bloc de Notas y la ventana de Símbolo del sistema. Luego, reinicia el proceso de instalación de Windows. El error "<ProductKey>" debería haber desaparecido y podrás continuar con la instalación normalmente.

Conclusión

El error "<ProductKey>" en WinPE es un inconveniente menor que se resuelve fácilmente verificando la presencia y el contenido del archivo `ei.cfg`. Con estos pasos, podrás superar este obstáculo y disfrutar de una instalación de Windows fluida con BCDeX.
