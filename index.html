Funcionalidades Principales de BCDeX

⚙️

Gestión de Particiones Segura

Utiliza herramientas nativas de Windows (`diskpart.exe`) para crear una partición temporal y dedicada, asegurando un espacio aislado para los archivos de instalación del sistema operativo.

⚡

Automatización BCD y Scripts

Ejecuta de forma automática y controlada scripts de PowerShell para el montaje de imágenes ISO y la configuración precisa de los datos de arranque (BCD) del sistema.

📁

Extracción y Copia Optimizada

Extrae únicamente los archivos esenciales necesarios para el inicio de la instalación desde su ISO de Windows (original o modificada), optimizando el tiempo de preparación y el uso del espacio en disco.

⚠️

Seguridad: Aclaración sobre Falsos Positivos

Debido a la manipulación necesaria de la configuración de arranque (BCD) y particiones, algunos antivirus pueden reportar a BCDeX como un "falso positivo". Aseguramos que la herramienta es completamente segura y solo ejecuta comandos fundamentales de Windows para su funcionalidad.
