🛠️ Guía: Instalar Windows desde una partición con BCDeX

Olvídate del USB. Con BCDeX puedes instalar Windows directamente desde el disco duro. Aquí te explicamos cómo hacerlo paso a paso, sin complicaciones.

El proceso con BCDeX: Fácil y Rápido

Instalar Windows desde una partición del disco duro es un método increíblemente eficiente que BCDeX automatiza por completo. El proceso comienza seleccionando el disco donde se creará una partición temporal y la imagen ISO de Windows que deseas instalar. BCDeX es compatible con una amplia gama de versiones, desde Windows 7 hasta Windows 11, incluidas las ISO modificadas. Una de sus grandes ventajas es que omite automáticamente las comprobaciones de TPM y Secure Boot, facilitando la instalación de Windows 11 en hardware no compatible.

Una vez que presionas "Siguiente", la aplicación se encarga de todo: crea la partición, monta la imagen ISO y copia los archivos de instalación. Es un proceso totalmente desatendido hasta el siguiente paso.

Personalización y Optimización (Opcional)

Antes de reiniciar, BCDeX te ofrece una de sus funciones más potentes: la personalización de la instalación. Puedes crear un usuario local, evitar la configuración de red durante el OOBE (Out-of-Box Experience), eliminar la telemetría y desinstalar el bloatware preinstalado. Esto te permite tener un sistema operativo más limpio, rápido y privado desde el primer arranque.

Instalación y Limpieza Final

Con todo preparado, la aplicación te pedirá reiniciar el equipo. Al hacerlo, el sistema arrancará automáticamente en el entorno de instalación de Windows que BCDeX ha preparado. Si por alguna razón no lo hace, simplemente selecciona la entrada "BCDeX Boot" desde el menú de arranque de tu PC (Boot Menu).

💡 Importante:

Durante el proceso de instalación de Windows, verás una partición llamada "BCDeX Vol". Es crucial que no la elimines, ya que contiene todos los archivos necesarios para completar la instalación.

Una vez que tu nuevo Windows esté instalado y funcionando, el último paso es la limpieza. Vuelve a ejecutar BCDeX (puedes encontrarlo en la partición temporal) y selecciona la opción "Deshacer cambios". Esto eliminará la partición de instalación, liberando ese espacio y dejando tu sistema impecable, como si nada hubiera pasado.
